Northwestern Football Website

A website for fans to track Northwestern's 2024 schedule and express interest in buying tickets.

Code

Overview

For my Web Development class, I designed and developed a responsive web page for fans to view Northwestern Football’s 2024 home game schedule and sign up to be notified when tickets go on sale. The page features a clean and visually appealing layout inspired by the official Northwestern football website. A key component of the project is an interactive image carousel on the landing page, allowing users to scroll through pictures smoothly using buttons or keyboard shortcuts. The form includes various input types, including a price slider, and provides real-time validation feedback to ensure proper user input.

My Impact

I challenged myself to improve both my front-end development and user experience design skills. I implemented an intuitive carousel with smooth transitions, ensuring users could navigate the page effortlessly. I prioritized accessibility by enabling keyboard navigation, ensuring proper color contrast, utilizing semantic HTML elements, and using ARIA attributes to enhance usability for screen readers. Additionally, I refined my approach to form validation, creating clear and helpful error messages when users attempt to submit incomplete forms. I laid a strong foundation for expanding the website in the future, with potential additions like game-specific pages, stadium maps, and team merchandise integration.
